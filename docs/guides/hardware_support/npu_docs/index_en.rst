####################
Ascend NPU Chip for PaddlePaddle Operation
####################

Huawei Ascend 910 is an AI processor enjoying superior computing power. The Paddle NPU version supports the model training and inference on the Kunpeng CPU and the Ascend NPU of Huawei. 

Following guides can help you better know and experience how to run PaddlePaddle on the Ascend chip:

- `Instruction of installing the Paddle Ascend GPU version <./paddle_install_cn.html>`_ 
- `Example of training on the Paddle Ascend NPU version <./train_example_cn.html>`_
