Hardware Support
##########

You can know the hardware support of the PaddlePaddle framework by reading the following information:

- `Hardware support of PaddlePaddle <./hardware_info_cn.html>`_ : A description of the hardware supported by PaddlePaddle
- `Kunlun XPU chip used to run PaddlePaddle <./xpu_docs/index_cn.html>`_ : An instruction of how to install an run PaddlePaddle in the Kunlun XPU chip
- `Hygon DCU chip used to run PaddlePaddle <./rocm_docs/index_cn.html>`_ : A description of how to install and run PaddlePaddle in the Hygon DCU chip
- `HUAWEI Ascend NPU chip used to run PaddlePaddle <./npu_docs/index_cn.html>`_ : A description of how to install and run PaddlePaddle in the Ascend NPU chip
- `Graphcore IPU chip used to run PaddlePaddle <./ipu_docs/index_cn.html>`_ : A description of how to install and run PaddlePaddle in the IPU chip
